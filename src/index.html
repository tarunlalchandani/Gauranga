Introduction
An intuit service

A blockchain based autonomous decentralized network which facilitates the trade of old (used) bicycles. This web app is a simple demo to illustrate how decentralized network gets affected. This demo shows how to make a real-world blockchain end-user client working and how to sign transactions (records to be posted on blockchain) with a local Ethereum Wallet.
